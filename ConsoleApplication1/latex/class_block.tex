\doxysubsection{Подробное описание}
Абстрактный базовый класс "{}Блок обработки сигналов"{}. 


\begin{DoxyItemize}
\item 
\end{DoxyItemize}

Определяет общий интерфейс для всех узлов системы обработки (фильтров, сумматоров и т.\+д.). Содержит общие методы для работы с именем блока, обработки данных и сброса внутреннего состояния.
