\doxysubsection{Подробное описание}
Экспортируемый C-\/совместимый интерфейс (API) библиотеки. 

Предоставляет набор функций-\/оберток для работы с C++ классом \doxylink{class_processing_system}{Processing\+System} из других языков программирования (например, C\#, Python, C) через динамическую библиотеку (DLL).
