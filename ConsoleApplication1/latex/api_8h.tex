\doxysubsection{Подробное описание}
Экспортируемый C-\/совместимый интерфейс (API) библиотеки. 

Предоставляет набор функций-\/оберток для работы с C++ классом \doxylink{class_processing_system}{Processing\+System} из других языков программирования (например, C\#, Python, C) через динамическую библиотеку (DLL).

Создает новый экземпляр \doxylink{class_processing_system}{Processing\+System}. 

Получает текст последней перехваченной ошибки (Exception).

Обрабатывает целый массив данных (сигнал) через заданный блок.

Сбрасывает состояние буферов памяти всех блоков в системе.

Вычисляет текущий выходной сигнал конкретного блока.

Соединяет выходные порты нескольких блоков с входом целевого блока.

Добавляет блок сумматора в систему.

Добавляет БИХ-\/фильтр (IIR) в систему.

Добавляет КИХ-\/фильтр (FIR) в систему.

Уничтожает экземпляр системы и освобождает память.
