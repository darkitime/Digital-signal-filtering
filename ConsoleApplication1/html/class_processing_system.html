Подробное описание

Система управления графом обработки сигналов.

Позволяет регистрировать различные блоки (фильтры, сумматоры), устанавливать связи между ними (кто откуда берет данные) и выполнять комплексные вычисления над сигналом с автоматическим разрешением зависимостей.
