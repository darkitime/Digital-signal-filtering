Подробное описание

Экспортируемый C-совместимый интерфейс (API) библиотеки.

Предоставляет набор функций-оберток для работы с C++ классом ProcessingSystem из других языков программирования (например, C#, Python, C) через динамическую библиотеку (DLL).

◆ getLastError()

API_EXPORT const char * getLastError ( )

Получает текст последней перехваченной ошибки (Exception).

Если функции API сталкиваются с C++ исключениями, они сохраняются во внутренний буфер.

Возвращает
Указатель на C-строку с текстом ошибки или nullptr, если ошибок не было.
