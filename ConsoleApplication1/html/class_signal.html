Подробное описание

Класс для представления и математической обработки одномерных сигналов.

Хранит динамический массив отсчетов сигнала и предоставляет базовые операции, такие как сложение, умножение на скаляр и конкатенация.

◆ operator+()

Signal Signal::operator+ ( const Signal & other) const
inline

Оператор сложения двух сигналов.

Если сигналы разной длины, размер результата равен максимальной длине. Недостающие отсчеты короткого сигнала считаются равными 0.

Аргументы
otherВторой сигнал для сложения.
Возвращает
Новый объект сигнала, содержащий покомпонентную сумму.
